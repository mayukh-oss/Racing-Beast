## Notes
1. All image files should be in PNG or JPG format as specified
2. Audio files should be in MP3 format
3. Car images (car_1.png through car_41.png) need to be created/sourced for complete game functionality
4. Background images should match the display resolution (800x600 or higher)
5. Score files will be automatically generated by the game

## Recommended Asset Specifications
- **Screen Resolution**: 800x600 pixels
- **Car Sprites**: 50x100 pixels (transparent background recommended)
- **Powerup Sprites**: 50x50 pixels (transparent background)
- **Background Images**: Minimum 800x600 pixels
- **Audio Format**: MP3, reasonable file sizes for quick loading
